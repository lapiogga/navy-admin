타입 정의:
```typescript
export type KnowledgeStatus = 'pending' | 'approved' | 'rejected' | 'hidden'

export interface Knowledge extends Record<string, unknown> {
  id: string
  title: string
  category: string        // 지식유형: 업무지식/기술지식/행정지식/법규지식/기타
  source: string          // 출처: 생산/카피
  keywords: string[]      // 키워드 배열
  content: string         // 지식 내용
  authorName: string
  authorUnit: string
  viewCount: number
  recommendCount: number
  rating: number          // 평균 평점 1~5
  ratingCount: number     // 평점 참여 수
  status: KnowledgeStatus
  isFavorite: boolean
  attachments: string[]   // 첨부파일 URL 배열
  createdAt: string       // YYYY-MM-DD
}

export interface KnowledgeComment extends Record<string, unknown> {
  id: string
  knowledgeId: string
  authorName: string
  content: string
  createdAt: string
}
```

- POST /sys13/knowledge/:id/favorite — 즐겨찾기 토글
- PUT /sys13/knowledge/:id/status — 관리자 상태변경 (body: { status: KnowledgeStatus })
- GET /sys13/knowledge/:id/comments — 댓글 목록

- DELETE /sys13/knowledge/:id/comments/:commentId — 댓글 삭제
- GET /sys13/stats/category — 유형별 통계 (반환: [{category, count, ratio}])
