# 초과근무관리체계 (SYS01) 요구사항 정의서

## 1. 시스템 개요

| 항목 | 내용 |
|------|------|
| 시스템명 | 초과근무관리체계 |
| 시스템 코드 | SYS01 |
| 복잡도 | 상 |
| 총 단위프로세스 | 99개 |
| 대메뉴 수 | 7개 (신청서관리, 현황조회, 부대관리, 당직업무, 개인설정, 게시판, 관리자) |
| 소메뉴 수 | 28개 |

### 1.1 시스템 목적

해병대 장병의 초과근무 신청, 결재, 현황 조회, 월말결산을 통합 관리하며, 당직업무 및 부대관리 기능을 포함하여 초과근무 행정업무의 효율성을 높인다.

### 1.2 사용자 역할

| 역할 | 설명 | 주요 기능 |
|------|------|----------|
| 일반 사용자 | 초과근무 신청자 | 신청서 작성, 나의 근무현황 조회, 부재관리, 개인설정 |
| 결재자 | 초과근무 승인권자 | 신청서 결재(승인/반려), 일괄처리 승인 |
| 부대 관리자 | 부대별 관리 담당자 | 부대 근무현황, 부대인원 조회, 최대인정시간/근무시간/공휴일 관리 |
| 당직근무자 | 당직 담당자 | 초과근무자 확인, 당직개소 관리 |
| 체계 관리자 | 시스템 관리자 | 월말결산 마감기한 설정, 관리자 메뉴 전체 |
