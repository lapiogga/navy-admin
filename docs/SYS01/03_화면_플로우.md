# 초과근무관리체계 (SYS01) 화면 플로우

## 1. 시스템 진입 플로우

```
[메인 포탈 로그인]
    │
    ▼
[메인 대시보드]
    │
    ├── "초과근무관리체계" 클릭
    │
    ▼
[SYS01 홈] (/sys01)
    │
    ├── 사이드바 메뉴 선택
    │
    ▼
[각 기능 화면]
```

**세션 관리:**
- 창 닫기 / exit → 메인 포탈로 이동
- 세션 만료 → 로그인 화면으로 이동

## 3. 일괄처리 → 승인 플로우

```
[일괄처리] (/sys01/1/3)
    │

    │
    ├── [일괄 신청] 클릭 (선택된 3명)
    │       │
    │       ▼
    │   ┌──────────────────┐
    │   │ 일괄 신청 Modal    │
    │   │ ・구분(사전/사후)    │
    │   │ ・신청일            │
    │   │ ・근무기간          │
    │   │ ・근무시간          │
    │   │ ・근무사유          │
    │   │ ・첨부파일          │
    │   │     [저장]         │
    │   └────────┬─────────┘
    │            │ status=pending
    │            ▼
    │   하단: 일괄처리 현황 테이블에 추가
    │

    ├── [승인] → status=approved
    └── [반려] → 사유 입력 → status=rejected
```
