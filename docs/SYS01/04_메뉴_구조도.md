### 2.2 컴포넌트 재사용 관계

```
OtUnitPersonnelPage ──→ /sys01/2/7 (자료 출력)
                    ──→ /sys01/3/1 (부대인원 조회)
                    (동일 컴포넌트, 2개 URL에서 재사용)

OtUnitStatusPage ──→ /sys01/2/3 (부대 근무 현황)  [탭 1]
                 ──→ /sys01/2/4 (부대 근무 통계)  [탭 2]
                 ──→ /sys01/2/5 (부대 부재 현황)  [탭 3]
                 (1개 컴포넌트, 3개 URL → 탭으로 분리)

SimpleBoardPage ──→ /sys01/6/1 (공지사항, sysCode=sys01-notice)
                ──→ /sys01/6/2 (질의응답, sysCode=sys01-qna)
                (Phase 1 공통게시판 컴포넌트 재사용)

AdminRoutes ──→ /sys01/admin/* (관리자 전체)
            (Phase 1 공통기능 컴포넌트 재사용)
```

## 4. URL 규칙

| 패턴 | 설명 | 예시 |
|------|------|------|
| `/sys01` | SYS01 홈 (서브시스템 랜딩) | SubsystemHomePage |
| `/sys01/{대메뉴}/{소메뉴}` | 일반 메뉴 | /sys01/1/1 (신청서 작성) |
| `/sys01/admin/{기능}` | 관리자 메뉴 | /sys01/admin/system-mgr |

**대메뉴 번호 규칙:**
- 1: 신청서 관리
- 2: 현황조회
- 3: 부대관리
- 4: 당직업무
- 5: 개인설정
- 6: 게시판
- admin: 관리자
