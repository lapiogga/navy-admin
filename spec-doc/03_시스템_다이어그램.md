## 2. C4 Container 다이어그램 (Level 2)

시스템 내부 컨테이너 구성. 현재 MVP 단계에서는 SPA + MSW만 운영 중.

```mermaid
C4Container
    title 해병대 행정포탈 컨테이너 구조

    Person(user, "사용자")

    System_Boundary(portal, "해병대 행정포탈") {
        Container(spa, "SPA 프론트엔드", "React 18, TypeScript 5, Vite 5", "18개 서브시스템 UI\n213개 tsx 페이지\nAnt Design 5 ProLayout")
        Container(state, "상태 관리", "Zustand 5 + TanStack Query 5", "authStore: 인증/세션\nuiStore: UI 상태\nQuery Cache: 서버 데이터")
        Container(msw, "Mock Service Worker", "MSW 2.x", "네트워크 레벨 API 모킹\n18개 서브시스템 + auth + common 핸들러\nFaker.js 9 데이터 생성")
        Container(api, "REST API", "Spring Boot, Java", "비즈니스 로직 (추후 개발)")
        ContainerDb(db, "PostgreSQL", "RDBMS", "업무 데이터 (추후 개발)")
    }

    Rel(user, spa, "HTTPS")
    Rel(spa, state, "상태 읽기/쓰기")
    Rel(state, msw, "fetch (Axios)")
    Rel(msw, api, "전환 예정", "HTTP")
    Rel(api, db, "JDBC")
```

## 5. 인증/세션 흐름 다이어그램

Zustand persist 기반 세션 관리. 30분 만료, 크로스탭 로그아웃 지원.

```mermaid
sequenceDiagram
    actor U as 사용자
    participant L as LoginPage
    participant P as PortalPage (메인)
    participant S as SubsystemProLayout
    participant A as authStore (Zustand)
    participant M as MSW Handler

    U->>L: /login 접속
    L->>A: login(credentials)
    A->>M: POST /auth/login
    M-->>A: { token, user }
    A-->>A: persist to localStorage<br/>sessionExpiry = now + 30min
    A-->>L: 인증 성공
    L->>P: navigate('/')

    U->>P: 서브시스템 카드 클릭
    P->>P: window.open('/sys01')
    P->>S: RequireAuth 검증

    Note over S: SubsystemProLayout 진입<br/>ProLayout 사이드바 + 헤더<br/>SUBSYSTEM_MENUS[sysCode]

    S->>A: checkSession()
    A-->>S: 유효 (expiry > now)

    Note over S: 업무 수행 중...

    alt 세션 만료 (30분)
        S->>A: checkSession()
        A-->>A: expiry <= now
        A-->>S: 세션 무효
        S->>L: navigate('/login')
    else 포탈 복귀 (Exit 버튼)
        S->>P: window.opener.focus()
        S->>S: window.close()
    else 로그아웃
        S->>A: logout()
        A-->>A: clear persist
        S->>S: window.close() (opener 있으면)
        S->>L: navigate('/login') (없으면)
    end
```
